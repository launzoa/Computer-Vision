* **Filtro da Média (Mean Filter):** Aplica uma suavização espacial linear. O algoritmo percorre a imagem calculando a média aritmética dos canais BGR de todos os pixels englobados pela área do *kernel*, atribuindo esse valor resultante ao pixel central. É um método base para a redução de ruídos, embora tenha como característica o borramento das bordas estruturais da imagem.
* **Filtro da Mediana (Median Filter):** Aplica uma filtragem espacial estatística (não-linear). O algoritmo extrai a vizinhança do pixel, ordena os valores de intensidade de cada canal de cor utilizando uma função de ordenação (`sort_pixels`), e substitui o pixel central pelo valor estritamente mediano. Este filtro destaca-se principalmente por ser altamente eficaz na remoção de ruídos impulsivos (como o ruído "sal e pimenta"), preservando ao máximo a nitidez das bordas originais.
